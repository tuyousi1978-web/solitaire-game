# ソリティア（クロンダイク）

美しいデザインのソリティアゲームです。iPhone/iPadでの横向き表示に最適化されています。

## 特徴

- 📱 モバイル対応（iPhone/iPad横向き表示最適化）
- 🎴 大きくて見やすいカード表示
- ⚡ スムーズなドラッグ&ドロップ操作
- 🎯 ダブルクリックで自動移動
- 📊 スコアとタイマー機能
- ✨ 美しいグラデーション背景

## 遊び方

### 基本操作
- **山札をクリック**: 山札から3枚めくる
- **カードをドラッグ**: カードを移動
- **ダブルクリック**: ファウンデーションへ自動移動
- **新しいゲーム**: 左下のボタンでゲームをリセット

### ルール
1. **テーブロー**: 降順で色を交互に並べる（赤→黒→赤...）
2. **ファウンデーション**: スート別に昇順で並べる（A→K）
3. **勝利条件**: すべてのカードをファウンデーションに移動

## GitHub Pagesでの公開方法

1. このリポジトリをGitHubにプッシュ
2. リポジトリの設定 → Pages → Source: main branch
3. 公開されたURLにアクセス

## ローカルでの実行

ブラウザで `index.html` を開くだけで動作します。

```bash
# ダウンロード
git clone https://github.com/yourusername/solitaire-game.git
cd solitaire-game

# ブラウザで開く
open index.html  # macOS
start index.html # Windows
```
