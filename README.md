# ソリティア（クロンダイク）

美しいデザインのソリティアゲームです。iPhone/iPadでの横向き表示に最適化されています。

## 特徴

- 📱 **モバイル完全対応** - iPhone/iPad横向き表示最適化
- 👆 **タップ操作対応** - タップで選択＆移動（iPhone最適化）
- 🎴 **大きくて見やすいカード** - 300px × 420px の大型カード
- ⚡ **スムーズな操作** - ドラッグ&ドロップとタップの両方に対応
- 🎯 **ダブルタップ自動移動** - カードをダブルタップでファウンデーションへ
- 📊 **スコア＆タイマー** - ゲームの進行を記録
- ✨ **美しいデザイン** - グラデーション背景と洗練されたUI

## 操作方法

### iPhone/iPad（タップ操作）
1. **カードを選択**: 移動したいカードをタップ（金色の枠が表示されます）
2. **移動先を選択**: 移動先の場所をタップ
3. **ダブルタップ**: カードをダブルタップでファウンデーションへ自動移動
4. **選択解除**: 背景をタップで選択解除
5. **山札**: 山札をタップして3枚めくる

### PC（ドラッグ＆ドロップ）
- **カードをドラッグ**: カードを移動
- **ダブルクリック**: ファウンデーションへ自動移動
- **山札をクリック**: 山札から3枚めくる

### 共通
- **新しいゲーム**: 左下のボタンでゲームをリセット

## ゲームルール

### 基本ルール
1. **テーブロー（下の7列）**: 降順で色を交互に並べる
   - 例: 黒のキング → 赤のクイーン → 黒のジャック...
2. **ファウンデーション（右上の4箇所）**: スート別に昇順で並べる
   - 各スート（♠♥♣♦）ごとにAからKまで
3. **勝利条件**: すべてのカードをファウンデーションに移動

### ポイント
- ファウンデーションにカードを移動: +10点
- テーブローにカードを移動: +5点
- 山札から引く: -2点

## カスタマイズ

カードサイズや色を変更したい場合は、`index.html` 内の以下のCSSを編集してください：

```css
.card {
    width: 300px;  /* カードの幅 */
    height: 420px; /* カードの高さ */
}

body {
    background: linear-gradient(135deg, #1a472a 0%, #2d5a3d 50%, #0d2818 100%);
}
```
